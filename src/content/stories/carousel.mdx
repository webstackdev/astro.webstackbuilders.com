# Carousel Component

The Carousel component provides an interactive way to display multiple items with navigation controls and smooth transitions.

import Carousel from '@components/Carousel/index.astro'

## Basic Carousel

<Carousel items={[
  { title: "First Slide", content: "Content for the first slide" },
  { title: "Second Slide", content: "Content for the second slide" },
  { title: "Third Slide", content: "Content for the third slide" }
]} />

## Usage

```astro
---
import Carousel from '@components/Carousel/index.astro'

const slides = [
  { title: "Slide 1", content: "Your content here" },
  { title: "Slide 2", content: "Another slide" }
]
---

<Carousel items={slides} />
```

## Features

- **Touch Support**: Swipe gestures on mobile devices
- **Navigation**: Previous/next buttons and pagination dots
- **Auto-play**: Optional automatic slide progression
- **Responsive**: Adapts to different screen sizes
- **Accessibility**: Keyboard navigation and screen reader support

## Props

- **items**: Array of slide objects with title, content, and optional image
- **autoPlay** (optional): Enable automatic slide progression
- **interval** (optional): Time between slides in milliseconds
- **showDots** (optional): Display pagination dots
- **showArrows** (optional): Display navigation arrows
