# Contact Form Component

The ContactForm component provides a comprehensive form for users to send inquiries with validation and error handling.

import ContactForm from '@components/ContactForm/index.astro'

## Default Contact Form

<ContactForm />

## Usage

The ContactForm component includes:
- Name and email validation
- Message length requirements
- GDPR consent checkbox
- Error handling and success states
- Accessibility features

```astro
---
import ContactForm from '@components/ContactForm/index.astro'
---

<ContactForm />
```

## Features

- **Validation**: Client-side and server-side validation
- **Accessibility**: Proper labels, ARIA attributes, and keyboard navigation
- **GDPR Compliant**: Optional consent checkbox
- **Error Handling**: Clear error messages for all fields
- **Success States**: Confirmation after successful submission
