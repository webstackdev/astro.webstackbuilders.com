# Hero Component

The Hero component is used to create impactful landing page headers with various layouts and content options.

import Hero from '@components/Hero/index.astro'

## Default Hero

<Hero
  title="Welcome to Our Platform"
  subtitle="Build amazing applications with our comprehensive toolkit"
  variant="default"
/>

## Usage

```astro
---
import Hero from '@components/Hero/index.astro'
---

<Hero
  title="Your Title"
  subtitle="Your subtitle"
  variant="default"
/>
```
