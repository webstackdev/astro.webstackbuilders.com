## Usage

```astro
---
import Testimonials from '@components/Testimonials/index.astro'

const testimonials = [
  {
    name: "Customer Name",
    company: "Company Name",
    role: "Job Title",
    content: "Testimonial content here...",
    avatar: "/path/to/avatar.jpg",
    rating: 5
  }
]
---

<Testimonials testimonials={testimonials} />
```

## Props

- **testimonials**: Array of testimonial objects
- **layout** (optional): Display layout - "grid" (default) or "carousel"
- **showRating** (optional): Display star ratings
- **maxItems** (optional): Maximum number of testimonials to display
