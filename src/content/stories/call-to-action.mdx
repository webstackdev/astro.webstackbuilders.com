# Call to Action Component

The CallToAction component provides flexible call-to-action sections with customizable styling and layouts.

import CallToAction from '@components/CallToAction/index.astro'

## Primary CTA

<CallToAction
  headline="Ready to Get Started?"
  text="Transform your business with our expert web development services."
  primaryButton={{
    text: "Start Your Project",
    href: "/contact"
  }}
/>

## CTA with Secondary Button

<CallToAction
  headline="Need More Information?"
  text="Explore our comprehensive services and see how we can help your business grow."
  primaryButton={{
    text: "View Services",
    href: "/services"
  }}
  secondaryButton={{
    text: "Learn More",
    href: "/about"
  }}
/>

## Minimal CTA

<CallToAction
  text="Subscribe to our newsletter for the latest updates."
  primaryButton={{
    text: "Subscribe Now",
    href: "/newsletter"
  }}
/>

## Usage

```astro
---
import CallToAction from '@components/CallToAction/index.astro'
---

<!-- Basic CTA -->
<CallToAction
  headline="Your Headline Here"
  text="Your compelling message."
  primaryButton={{
    text: "Button Text",
    href: "/your-link"
  }}
/>

<!-- CTA with both buttons -->
<CallToAction
  headline="Your Headline Here"
  text="Your compelling message."
  primaryButton={{
    text: "Primary Action",
    href: "/primary-link"
  }}
  secondaryButton={{
    text: "Secondary Action",
    href: "/secondary-link"
  }}
/>
```

## Props

- **headline** (optional): Main headline text
- **text**: Supporting text or description
- **primaryButton**: Primary button configuration (text, href)
- **secondaryButton** (optional): Secondary button configuration
