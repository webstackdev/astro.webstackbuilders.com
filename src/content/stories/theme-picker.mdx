# Theme Picker Component

The ThemePicker component allows users to switch between different visual themes of the website.

import ThemePicker from '@components/ThemePicker/index.astro'

## Default Theme Picker

<ThemePicker />

## Theme Picker with Label

<ThemePicker showLabel={true} />

## Compact Theme Picker

<ThemePicker variant="compact" />

## Usage

The ThemePicker component provides:

- Light/dark mode toggle
- System preference detection
- Persistent theme selection
- Smooth transitions between themes
- Accessibility support

```astro
---
import ThemePicker from '@components/ThemePicker/index.astro'
---

<!-- Basic theme picker -->
<ThemePicker />

<!-- With visible label -->
<ThemePicker showLabel={true} />

<!-- Compact variant -->
<ThemePicker variant="compact" />
```

## Features

- **System Detection**: Automatically detects user's system theme preference
- **Persistence**: Remembers user's theme choice across sessions
- **Smooth Transitions**: Animated transitions between theme changes
- **Accessibility**: Proper ARIA labels and keyboard navigation
- **Multiple Variants**: Different visual styles available

## Props

- **showLabel** (optional): Display text label next to the toggle
- **variant** (optional): Visual variant - "default" or "compact"
- **position** (optional): Position of the picker - "header", "footer", or "inline"
