# Avatar Component

The Avatar component displays user profile images with fallback to initials when images are not available.

import Avatar from '@components/Avatar/index.astro'

## Default Avatar with Image

<Avatar
  src="/assets/images/avatar-example.jpg"
  alt="John Doe"
  name="John Doe"
/>

## Avatar with Initials Fallback

<Avatar
  name="Jane Smith"
  alt="Jane Smith"
/>

## Different Sizes

<Avatar
  name="Small Avatar"
  size="sm"
/>

<Avatar
  name="Medium Avatar"
  size="md"
/>

<Avatar
  name="Large Avatar"
  size="lg"
/>

## Usage

```astro
---
import Avatar from '@components/Avatar/index.astro'
---

<Avatar
  src="/path/to/image.jpg"
  alt="User Name"
  name="User Name"
  size="md"
/>
```
