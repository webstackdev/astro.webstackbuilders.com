# Social Links Component

The Social component displays social media links and sharing buttons with customizable styling and icons.

import Social from '@components/Social/index.astro'

## Social Media Links

<Social
  links={[
    { platform: "twitter", url: "https://twitter.com/webstackbuilders" },
    { platform: "linkedin", url: "https://linkedin.com/company/webstackbuilders" },
    { platform: "github", url: "https://github.com/webstackbuilders" },
    { platform: "youtube", url: "https://youtube.com/@webstackbuilders" }
  ]}
/>

## Social Sharing Buttons

<Social
  type="sharing"
  url="https://webstackbuilders.com"
  title="Check out this amazing website!"
  platforms={["twitter", "linkedin", "facebook", "email"]}
/>

## Vertical Layout

<Social
  links={[

    { platform: "linkedin", url: "https://linkedin.com/company/webstackbuilders" },

## Usage

```astro
---
import Social from '@components/Social/index.astro'
---

<!-- Social media links -->
<Social
  links={[
    { platform: "twitter", url: "https://twitter.com/yourhandle" },
    { platform: "linkedin", url: "https://linkedin.com/company/yourcompany" }
  ]}
/>

<!-- Social sharing -->
<Social
  type="sharing"
  url="https://yoursite.com"
  title="Share this page"
  platforms={["twitter", "linkedin", "facebook"]}
/>
```

## Features

- **Multiple Platforms**: Support for major social media platforms
- **Sharing Integration**: Built-in social sharing functionality
- **Customizable Icons**: SVG icons with consistent styling
- **Responsive Design**: Adapts to different screen sizes
- **Accessibility**: Proper labels and keyboard navigation

## Props

- **links**: Array of social media links (for profile links)
- **type**: "links" (default) or "sharing"
- **url**: URL to share (for sharing type)
- **title**: Title to share (for sharing type)
- **platforms**: Array of platform names to include
- **layout**: "horizontal" (default) or "vertical"
- **size**: Icon size - "small", "medium" (default), or "large"
